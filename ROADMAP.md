## 🔄 종목/리그 전환 뷰
- 캘린더 및 피드 전체를 **종목(리그) 단위로 전환**할 수 있는 탭/선택 UI
- 선택한 종목에 따라 일정, 리뷰 피드, 대시보드 통계가 모두 연동

## ❤️ 응원팀 설정
- 프로필에서 **내 응원팀**을 사전 설정 (예: KBO → 두산, K리그 → 전북)
- 리뷰 작성 시 자동 선택 / 경기 필터링 / 맞춤 알림 기반으로 활용

## 🚀 기능 개발 예정
- **⚾ 경기 선택 리뷰** — 하루 여러 경기 중 리뷰할 경기를 직접 선택 (KBO 시즌 5경기/일 대비)
- **📸 직관 사진 업로드** — 리뷰에 인증샷 첨부
- **📊 구장별/팀별 상세 통계** — 내가 어느 구장 가면 지는지 과학적 증명
- **🏆 유저 랭킹 시스템** — 이번 주 직관왕, 리뷰왕 등 서열 정리
- **🔗 카카오톡/링크 공유** — 내 리뷰 & 대시보드 공유
- **💬 리뷰 댓글 기능** — 다른 팬들과 소통
- **📱 PWA 지원** — 홈화면 추가로 앱처럼 사용

## 🏛️ 명경기(대첩) 아카이브
- 나무위키 '대첩' 문서 기반으로 **역대 명경기 데이터를 사전 등록**
- 과거 경기에 대해서도 리뷰를 남길 수 있도록 지원 (그날의 추억 기록)
- 예시: 10.8 대첩, 잠실 역전극, 한일전 명경기 등

## 📩 경기 추가 요청
- 사용자가 DB에 없는 **과거 경기의 등록을 요청**할 수 있는 기능
- 관리자 승인 후 등록 → 해당 경기에 리뷰 작성 가능

## 🛡️ DB 스키마 안정성 강화
현재 다수의 컬럼이 자유 텍스트(TEXT) 기반이라 오타나 불일치 위험이 있음. 단계적으로 개선 예정.

| 현재 | 개선 방향 |
|------|----------|
| `status TEXT` (예정/진행중/종료) | `ENUM` 타입 또는 CHECK 제약조건 |
| `location TEXT` (직관/집관/펍) | `ENUM` 또는 정규화된 참조 테이블 |
| `home_team`, `away_team TEXT` | `teams` 마스터 테이블 + FK 참조 |
| `supporting_team TEXT` | `teams` FK 연동 |
| `stadium TEXT` | `stadiums` 마스터 테이블 + FK 참조 |

> 종목 확장 시점에 `leagues`, `teams`, `stadiums` 마스터 테이블을 도입하여 데이터 정합성을 확보할 계획.
