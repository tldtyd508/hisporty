WBC 개막이라 기념으로 그동안 생각만 하던 앱 하나 깎아왔어. 개발 잘 모르는 뉴비인데 요즘 AI 성능이 좋아서 '바이브 코딩'으로 퇴근하고 짬짬이 만들어봤음. 거창한 건 아니고 그냥 내가 쓰고 싶어서 만든 MVP 버전이야.

1. 경기 일정 및 KST 기준 시간
![Screenshot: Hero Section]

오늘 개막인 WBC 대한민국 경기랑 나머지 팀들 일정 확인 가능함.
시차 계산 귀찮아서 KST 기준으로 다 박아둠.
2. 0.5점 단위 별점 & 한줄평
![Screenshot: Review Card]

왓챠 같은 느낌으로 별점 남기고 한 줄로 기록하는 방식임.
응원하는 팀 선택해서 짧게 기록 남기기 좋게 만들었어.
3. 마이 대시보드
![Screenshot: Dashboard]

내가 쓴 리뷰 개수랑 승률(승요 지수) 같은 거 모아서 보여주는 섹션임.
로그아웃/게스트 모드도 되는데, 로그인하면 데이터 계속 쌓을 수 있음.
앞으로의 로드맵 (확장 계획)
일단 WBC로 시작했는데, 앞으로 종목이나 기능을 계속 늘려가 볼 생각이야.

종목 확장: WBC → KBO → K리그 → 배드민턴 → 월드컵 순으로 데이터 넣을 예정
기능 추가: 직관 사진 업로드, 구장별/팀별 상세 통계 분석
서버 고도화: 일단은 내가 쓰려고 만든 거라 피드백 주면 더 깎아보겠음
오픈 기념으로 데이터도 좀 넣어놨으니까 야구 좋아하는 형들은 한 번씩 구경해 줘.
