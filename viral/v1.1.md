---

저번에 WBC 개막 기념으로 바이브 코딩해서 앱 하나 만들었다고 글 올렸었는데
[이전 글 링크]

오늘 일본전 같이 보기로 한 사람들이 있어서 이따 개발일지 올릴 시간이 없을 거 같아서 지금 급하게 올림. 질 걸 알지만 그래도 응원은 해야지...


오늘 업데이트한 거 정리해봄.


1. 마이팀 설정 기능

![Screenshot: 마이팀 설정 바텀시트]

헤더에 마이팀 버튼 생겼음. 누르면 WBC 20개국 중에서 내 팀 고를 수 있음.
복수 선택 가능하고 풀별로 나눠놨음. 기본값은 당연히 대한민국.


2. 마이팀 경기 우선 표시

![Screenshot: 마이팀 카드 vs 다른 경기 카드]

하루에 경기가 8개씩 있는데 마이팀 경기를 맨 위에 고정시킴.
마이팀은 파란색 카드, 나머지는 회색으로 확실하게 구분되게 함.
나머지 경기는 "다른 7경기 더 보기"로 접어놨음.


3. 경기 카드 클릭하면 해당 경기 리뷰만 필터링

![Screenshot: 경기 선택 상태 + 리뷰 필터]

경기 카드를 탭하면 노란색 테두리 뜨면서 선택됨.
아래 리뷰 섹션이 해당 경기 리뷰만 보여줌.
다시 누르면 해제되고 전체 리뷰로 돌아감.


4. WBC 풀 스테이지 전 경기 데이터 입력

40개 전 경기 일정 다 넣었음. C조만 있던 거에서 A~D조 전체로 확장함.
종료된 경기는 스코어도 업데이트 해둠.


5. 리뷰 좋아요 기능

![Screenshot: 하트 버튼]

리뷰에 하트 달 수 있게 됨. 인기순 정렬도 추가함.



로드맵으로는 종목 확장(KBO 시즌 대비)이랑 직관 사진 업로드 같은 거 생각 중임.
일단 오늘은 일본전 응원하러 감. 져도 리뷰는 남기겠음 ㅋㅋ
